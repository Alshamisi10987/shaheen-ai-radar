# Shaheen | AI Adoption and Opportunity Radar

Shaheen (شاهين) is a single-page web dashboard for tracking AI adoption and sales opportunities across the UAE market. It scans companies, ministries, authorities, free zones, investors, and strategic partners, then scores each one on adoption level, buying intent, and product fit — surfacing the next best action to pursue a sale.

## Features

- **Opportunity Leaderboard** — ranked, filterable table of prospects with adoption, intent, and fit scores
- **Search & filters** — filter by emirate, sector, entity type, product, and score range
- **KPI overview** — at-a-glance metrics for market universe size, hot leads, product fit, and live signals
- **Live signal feed** — refreshable feed of partnership news and market activity
- **Prospect profiles** — detail modal per entity with a radar chart of product fit and a generated outreach message
- **Export tools** — export the filtered leaderboard, a single prospect profile, or sales-ready metrics
- **Product radar chart** — visual breakdown of adoption vs. fit across products

## Tech Stack

Built as a single self-contained HTML file — no build step, no dependencies beyond Google Fonts.

- HTML5 / CSS3 (custom properties, OKLCH color system)
- Vanilla JavaScript
- Inter & Space Grotesk typefaces

## Getting Started

No installation needed — it's a static page.

1. Clone the repo:
   ```bash
   git clone https://github.com/<your-username>/shaheen-ai-radar.git
   ```
2. Open `Shaheen-Radar-FINAL.html` directly in your browser, **or**
3. Enable [GitHub Pages](https://pages.github.com/) on this repo for a live hosted link.
